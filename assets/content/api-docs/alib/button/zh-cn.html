alibButton,alib-button

alib-button 是一个 Directive，但是也可以当作一个组件使用

Name Type Default Description
alibButton

按钮类型，类型为 'primary' | 'secondary' | 'danger'
alibType

和 alibButton 含义相同，一般使用 alibButton，为了减少参数输入, 设置按钮组件通过 alib-button 时，只能使用该参数控制类型
alibSize

按钮大小，目前可传的大小为：'xs' | 'sm' | 'md' | 'lg'
alibButtonIcon
Name Type Default Description
alibButtonIcon

按钮图标类型
